# Recognizing Unseen States of Unknown Objects by Leveraging Knowledge Graphs (WACV 2025)

## About
This repository contains the code and experimental evaluation material for the paper:
**Recognizing Unseen States of Unknown Objects by Leveraging Knowledge Graphs** (WACV 2025).

## Paper
The official paper can be found here:  
https://openaccess.thecvf.com/content/WACV2025/html/Gouidis_Recognizing_Unseen_States_of_Unknown_Objects_by_Leveraging_Knowledge_Graphs_WACV_2025_paper.html


## Evaluation

Our proposed method is implemented in the `OaSC` directory.

The `CompCos` directory contains the implementation of **CompCos** and additional baselines including **AOOP**, **TMN**, **SymNet**, and **LE+**.

The baseline methods  **ADE**, **CANET**, **IVER**, **KG-SP**, **OaDis** and **SCEN** are provided in their respective directories. 

Each directory includes a dedicated README with instructions for installation and execution.


## Dataset

Instructions for obtaining and preparing the datasets used in the paper are provided in the corresponding directories.


## Requirements

Dependencies are listed in the individual directories. Please refer to the README files inside each method folder.
